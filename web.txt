The Internet is a global network of computers that interconnect with each other.  Each computer connected to the Internet must have a unique address called an IP address.  It is like a home address for a computer.  When you connect to the Internet, your computer should have a permanent IP address.  Each computer needs software to communicate on the Internet.  The software is usually built into the computer's operating system.

When you click on a link, your browser and operating system figure out where you've clicked.  The web page that you are on has information associated with whatever you clicked on.  That information is called a Uniform resource locator or URL.  Your browser takes that URL, retrieves the website's name, and then uses the Domain Name System or DNS to get an Internet Protocol or IP address for the site.  Then, your browser delivers the following web page in HyperText Markup Language or HTML that your browser renders.
